# HypoDisc

On one dataset:

    hypodisc/run.py --depth 1:3 --min_support 2 --min_confidence 1 --multimodal tests/test_dataset_a.nt tests/test_dataset_b.nt
    
On two or more datasets:

    hypodisc/run.py --depth 1:3 --min_support 2 --min_confidence 1 --multimodal tests/test_dataset_a.nt tests/test_dataset_b.nt
